
# Automated Data Quality & Imputation Pipeline

## 📌 Overview
This project is an automated system for data cleaning, quality analysis, and missing value imputation. It helps improve dataset reliability for machine learning and data analysis.

## 🚀 Features
- Data quality checks
- Missing value detection
- Data cleaning
- Statistical imputation (mean, median, mode)
- Machine learning-based imputation (future scope)

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
